--- --- NXBoot Cydia Depiction

NXBoot enables starting custom boot code on the Nintendo Switch through the Fusée Gelée hardware exploit.

Works on iOS 9-16 and possibly later versions.

A Lightning OTG/camera adapter is required!

Please read the warning and disclaimer in this app's "Getting Started" section prior to using it to start custom boot code on your Nintendo Switch.
